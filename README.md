Simple and functional **To-Do List app** built with React.js.  
Demonstrates state management and CRUD with mock API.

## Features
- Add, delete, edit tasks
- Persist data with mock API
- Simple & clean UI

## Tech Stack
- Frontend: React.js
- API: Mock JSON server
- State: React hooks
